Class ImageAnalysis

Method for finding pixels connected to each other. Great for finding UI Elements on the screen based on pixel color filter. A good explanation of the Connected Component Analysis can be seen here: https://www.youtube.com/watch?v=ticZclUYy88 Uses a 4-connectivity 2-pass Hoshen-Kopelman algorithm
